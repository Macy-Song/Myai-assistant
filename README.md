# Spring AI Assistant 🤖

基于 **Spring AI** 框架开发的多模态 AI 助手系统。

## 项目介绍
这是一个基于 Spring AI 的智能 AI 助手，支持上下文对话和 AI 图像生成。通过提示词优化和异步请求处理，提升用户交互体验。

## 主要功能
- AI 智能对话
- 上下文记忆对话
- AI 图像生成
- 提示词优化
- 异步请求处理

## 技术栈
- **后端**：Java + Spring Boot + Spring AI
- **数据库**：MySQL
- **前端**：HTML + CSS + JavaScript
- **构建工具**：Maven
- **其他**：REST API

## 项目结构
- `pom.xml` — Maven 配置文件
- `src/main/java/` — Java 源代码
- `src/main/resources/static/` — 前端文件（HTML、CSS、JS）
- `src/main/resources/application.yml` — 配置文件
- `README.md` — 项目说明
